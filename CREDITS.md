# 素材与许可证

## UI 素材

- Paper 006，ambientCG：https://ambientcg.com/view?id=Paper006
  - 许可证：Creative Commons CC0 1.0 Universal
  - 用途：卡牌界面、事件面板与任务面板的羊皮纸背景。

- Settings，Lucide Icons：https://lucide.dev/icons/settings
  - 许可证：ISC License
  - 用途：设置按钮图标。

- Noto Sans SC，Noto Project：https://fonts.google.com/noto/specimen/Noto+Sans+SC
  - 许可证：SIL Open Font License 1.1
  - 用途：中英文 UI 字体。

以上素材均允许在本项目中使用和修改。字体许可证全文保存在 `Assets/Resources/Fonts/OFL.txt`。
