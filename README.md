# Zebra Digital Prototypes

Small, isolated feasibility prototypes for the Zebra final game project.

## Unity Version

Unity `6000.5.3f1`.

## Projects

1. `prototypes/01-card-turn-cycle/UnityProject`
   - Draw three cards from a seven-card deck.
   - End turn, discard the hand, and draw again.
   - Shuffle the discard pile into the deck when required.

2. `prototypes/02-card-play-reveal/UnityProject`
   - Click once to select and raise a card.
   - Click the selected card again to play it.
   - End turn to reveal the remaining cards one at a time.

Each `UnityProject` folder contains only `Assets`, `Packages`, and `ProjectSettings`. Add that exact folder through Unity Hub. Unity regenerates `Library`, `Logs`, `Temp`, and local user settings.
