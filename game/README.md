# Zebra Integrated Game

Unity version: `6000.5.3f1`

Open `UnityProject` as the Unity project root.

## Integrated Game V1

- Start round and draw five cards, with a maximum hand size of ten.
- Reshuffle the discard pile when the draw pile is empty.
- View the total deck, draw pile, and discard pile during the action phase.
- Buy gold royal cards into the discard pile.
- Delete a selected card from any owned-card zone using a scrollable deck view.
- Select a hand card, click it again, and choose a matching location.
- Place a minister and resolve Economy, Military, or Administration effects.
- End the round, reveal retained cards one by one, resolve retain effects, discard them, and start the next round.

The `integrated-game-v1` Git tag identifies the tested baseline before language and pointer-interaction upgrades.
